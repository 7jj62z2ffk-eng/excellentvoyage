## 发布选项

### 1. GitHub Pages (免费)
1. 创建 GitHub 账户
2. 创建新的仓库，例如 `sz-jialv-logistics.github.io`
3. 将此文件夹中的所有文件上传到仓库
4. 在仓库设置中启用 GitHub Pages
5. 网站将可通过 `https://yourusername.github.io/sz-jialv-logistics` 访问

### 2. Netlify (免费/付费)
1. 注册 Netlify 账户
2. 拖放此文件夹到 Netlify 控制面板
3. 配置域名（可选）
4. 网站将自动部署

### 3. 传统虚拟主机
1. 将所有 HTML 文件上传到主机的根目录
2. 确保 index.html 设为主页
3. 配置域名解析

### 4. AWS S3 (静态网站托管)
1. 创建 S3 存储桶
2. 启用静态网站托管
3. 上传所有文件
4. 配置路由规则
